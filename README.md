# ☀️🩺 어린이집 날씨·건강 변수 기반 업무 추천 시스템

## 주요 기능
### 1) 외부 환경 정보 실시간 수집

- 기상청 날씨 API
- 환경부 미세먼지·대기질 API
- 질병관리 관련 공개 데이터(API)

스케줄러를 이용해 주기적으로 호출하여 다음 정보를 저장한다.

- 오늘 날씨(기온, 강수 여부, 폭염·한파 특보 여부)
- 미세먼지/초미세먼지 단계(좋음/보통/나쁨/매우 나쁨)
- 주요 유행성 질병의 주의보/경보 여부

이 데이터가 나중에 “오늘의 체크리스트”를 만드는 기준이 된다.

### 2) 변수 기반 “오늘의 체크리스트” 자동 생성

외부 변수(날씨·대기질·질병 단계)와 규칙 기반 엔진을 사용해

그날 반드시 수행해야 할 건강·안전 업무를 자동으로 생성한다.

예시 1) 미세먼지 단계가 ‘나쁨’ 이상인 날

- 조건
    - PM10 또는 PM2.5가 설정한 기준 이상일 때
- 자동 생성 체크리스트
    - 실외 활동 취소 또는 일정 조정
    - 실내 공기 질 점검(공기청정기 작동 상태, 필터 확인 등)
    - 환기 방법·시간 조정(짧고 자주, 방향 설정 등)

예시 2) 독감 유행 주의보 발령 기간

- 조건
    - 감염병 동향 데이터에서 인플루엔자 주의보 이상일 때
- 자동 생성 체크리스트
    - 손 씻기·기침 예절 지도 강화
    - 공용 수건 대신 개인 수건·티슈 사용으로 전환
    - 교구·교실 소독 횟수 증가
